Política de Privacidad – AppOLO

Última actualización: 6 de abril de 2026

1. Información que recopilamos

  • Datos de cuenta: nombre de usuario y correo electrónico.
  • Datos de uso dentro de la aplicación.
  • Contenido generado como fotos, notas y registros.
  • Información del dispositivo.

2. Uso de la información

Utilizamos la información para:

  • Operar y mejorar la aplicación.
  • Gestionar procesos internos (despachos, aperturas).
  • Autenticación de usuarios.
  • Seguridad del sistema.

3. Almacenamiento de datos

Utilizamos servicios como Firebase (Google) para almacenar datos de forma segura.

4. Compartición de datos

No vendemos información personal. Solo se comparte cuando es necesario para operar la app o cumplir la ley.

5. Seguridad

Aplicamos medidas de seguridad para proteger los datos contra accesos no autorizados.

6. Retención de datos

Los datos se almacenan mientras sean necesarios para el funcionamiento de la app.

7. Derechos del usuario

El usuario puede solicitar acceso, corrección o eliminación de sus datos.

8. Permisos

  • Cámara (para fotos)
  • Almacenamiento
  • Internet

9. Cambios

Esta política puede actualizarse en cualquier momento.
